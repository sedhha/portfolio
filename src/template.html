My Water Chain

Manage WasteWater through BIot (BlockChain and IoT)

Ever wondered why BlockChain is all about currencies? Here We came up with yet another BlockChain implementation (Water Chain), where we are using Blockchain to monitor and store Water Stats generated by industrial users.

This is a full stack project integrated with IoT. I designed the entire project architecture using Raspberry Pi as the Brain of hardware control system and water sensors. Raspberry Pi creates it's local server where it pings to the backend database, that was built on Python (Flask). It communicates with the server to verify it's identity and then keeps monitoring the data and send the real time updates.

The front-end of the project was built using React JS that gives interactive dashboards and was also integrated using livepeer API that gives real time video streaming for process verification.

CoviFitness

Stay Fit this pandemic with this small fun pygame!

Due to extreme slowdown in the pandemic, we all have been used to staying on our beds all the time. Covi Fitness is a simple Virtual game that responds to your movements and using computer vision, it translates it as a virtual joystick to control the character's movement in the game!

Not just the fun part, the game UI/UX has been dsigned keeping in mind that the same can be used as an awareness spread for the kids about the prevention of covid-19 and the necessary steps to keep ourselves safe and healthy.

I simply used OpenCV (Python) and Pygame to build this game.

The Third Eye

See the World from the Eyes of Technology!

1.3 billion People have some sort of vision impairment. They face difficulties in simple day to day task like reading, recognizing faces, objects, etc. Despite the huge number surprisingly there are only a few devices in the market to aid them, which can be hard on the pocket. These devices essentially just magnify the images and only help those with mild to moderate impairment. There is no product in circulation for those who are completely blind.

The Third Eye brings a plethora of features at just 5% the cost. We set our minds to come up with a device that provides much more than just a sense of sight and most importantly is affordable to all. We see this product as an edge cutting technology for futuristic development of Assistive technologies.

Kratos Space Rover

Space Exploration Martian Rover Prototype

How many of you have heard about Curiosity? Well, Curiosity is one of the most popular space rovers built by NASA that explores the Martian environment and studies its geology and astrobiology in search of minerals and bio-signatures.

Similarly, I worked as an instrumentation and design engineer in building the entire architecture of our rover prototype that would perform various physical and biological tests to detect biosignatures in Martian-like Environment (MDRS, Utah, Hansville).

I worked on using micro-controller (Arduino-Nano & Uno) and interfaced them with the microprocessor (Raspberry Pi V3b+) via I2C and serial bus protocol. Moreover, I used socket programming for real-time image frame transmission and processed them using OpenCV to perform various computer vision analysis.
